# Lab 01: PyTorch Installation and Basic Tutorial Experiment

Course: Deep Learning and Computer Vision

## 1. Lab Goal

In this lab, you will install PyTorch, run official PyTorch tutorial walkthrough scripts, and complete one integrated FashionMNIST experiment.

Scripts `00_check_env.py` through `08_transfer_learning_preview.py` are walkthrough scripts based on official PyTorch tutorials. They help you understand installation verification, tensors, datasets, transforms, model building, autograd, training, saving/loading, and transfer learning preview.

Script `09_integrated_experiment.py` is the main integrated task for this lab. You will complete six numbered TODO blocks in this script and use its output in your final PDF submission.

## 4. Run the Lab Scripts

Open a terminal in the project root and enter the starter code directory:

```bash
cd labs/lab01_pytorch_intro/starter_code
```

Run the environment check first:

```bash
python 00_check_env.py
```

Run the walkthrough scripts during class:

```bash
python 01_tensors.py
python 02_data_tutorial.py
python 03_transforms.py
python 04_build_model.py
python 05_autograd.py
python 06_optimization.py
python 07_save_load_model.py
python 08_transfer_learning_preview.py
```

Then complete the six TODO blocks in the integrated experiment script and run:

```bash
python 09_integrated_experiment.py
```

You are encouraged to adjust the hyperparameters in the “Student Configuration Area” and observe how they affect the results.

### Integrated Experiment Script

`09_integrated_experiment.py` is the main integrated task. Complete only the six numbered TODO blocks in that file.

The six TODO blocks are:

1. Create train and test DataLoaders with `BATCH_SIZE`.
2. Complete the MLP model using `HIDDEN_DIM`.
3. Complete the SmallCNN model.
4. Train and test both MLP and CNN, then compare their test accuracy.
5. Save the selected model to `model_weights.pth`.
6. Complete the ResNet18 transfer learning preview.

The script must print clear output for the final screenshot.
