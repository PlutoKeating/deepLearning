• Object queries are learnable detection slots, not spatial positions.
• Each query attends globally to image features and predicts one object.
• Set prediction + bipartite matching enforces one-to-one assignment, eliminating NMS.
DETR replaces geometric anchors with learnable object queries, i.e., non-spatial detection slots.
Each query makes an independent prediction, while features are enriched through joint set-level 
interactions in the encoder-decoder.
